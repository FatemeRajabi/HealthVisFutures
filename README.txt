There are 5 main tabs included in this website. 
1. Home- the main page for Lab
2. Research- tab that includes all the projects 
3. Teaching- tab that includes courses taught
4. People- tab includes all the people contributing to this Lab
5. ResearchLab- description of other lab


The data is populated using json files in the respective html peges. Comments are given for the json file regarding the object for addition of data in future.
